# Монетка — Telegram Mini App

Премиальная игра «орёл или решка» в формате Telegram Mini App: ставка → выбор стороны →
подброс монеты. Серии множат выигрыш (×1.5 / ×2 / ×3), есть магазин скинов монет и фонов,
ежедневный бонус, достижения и лидерборд.

Всё приложение — один статический файл `index.html` (без сборки): чистый JS + SVG-монета,
Web Audio для звука, Telegram WebApp SDK, тактильная отдача (haptics).

## Возможности

- Чеканная SVG-монета с купольным металлом, фрезерованным гуртом и объёмным рельефом — 12 скинов
- 10 фонов-тем, магазин за внутриигровую валюту
- Серии побед с множителями, конфетти на выигрыш, шейк на проигрыш
- Provably fair + серверная валидация ставок (Supabase RPC)
- Лидерборд игроков, ежедневный бонус, достижения
- Свайп-навигация между панелями: Магазин · Игра · Задания · Лидеры

## Деплой

Сайт публикуется на **GitHub Pages** автоматически при пуше в `main`
(workflow `.github/workflows/deploy.yml`, Pages включается сам при первом запуске).

Живой адрес после первого деплоя:

```
https://egorify.github.io/MonetkaMiniApps/
```

Этот URL указывается в BotFather как Web App URL мини-приложения.

## Локальный запуск

Сборка не нужна — достаточно отдать `index.html` любым статическим сервером:

```bash
python3 -m http.server 8000
# открыть http://localhost:8000
```
